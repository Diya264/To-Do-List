# To-Do-List
The To-Do list application is used to allow users to manage task efficiently. I have developed this project using HTML, CSS and JavaScript. The application provides a user-friendly interface and is responsive to different screen size.

# Features
1. Add Task: Enter a task in the input field and click on 'Add' button to add it to list.
2. Edit Task: Click 'Edit' to edit the existing task and click on 'Save' button to save the changes.
3. Delete Task: Click on 'Delete' to delete the task from the list.
4. Status Of Task: Use checkboxes to mark the tasks Complete or Incomplete.
5. Local Storage: Task's are stored in browser's local storage to ensure that the task persist even after refreshing or closing the browser.

# Getting Started
1. Clone or download the repository.
2. Open index.html file in your web browser.

# Usage
* Enter a task in the input field and click on add button.
* Click on checkbox to mark task as complete or incomplete.
* Click 'Edit' to modify the existing tasks.
* Click 'Delete' to delete the task you no longer need.
